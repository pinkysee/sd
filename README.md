**СИТУАТИВНЫЕ КОДЫ.**  
  
CODE 0 - Офицер на земле и ему срочно требуется помощь. При данном коде, все не приоритетные действия нужно отложить и явится по данному коду. КОД объявляется когда офицер падает на землю из-за взаимодействия с гражданским, в таких ситуациях как открытый огонь по офицеру и он ранен, при том, что его сбили и т.д.  
CODE 1 - Офицер в бедственном положении и ему требуется помощь.. При данном коде, значит, что офицера кто-то преследует или его окружают и идет угроза его жизни. При данном КОДе каждый офицер должен отложить не приоритетные вызовы и задачи отложить и явится на место КОДа.  
CODE 2 - Приоритетный вызов. По этому КОДу принимаются вызовы 9-1-1, которые могут подождать и никакому гражданскому или окружающим ничего не угрожает. При данном КОДе, Вы двигаетесь без включенной сирены, но с включенной системой сликтоп (с стробоскопами, но без сирены). Вы соблюдаете полностью дорожный кодекс и не нарушаете его при данном КОДе.  
CODE 2 повышенный - означает приоритетный вызов, который не угрожает ничьей жизнь, но для решение задачи нужно поспешить. Это тот же КОД 2, но при этом Вы можете не соблюдать скоростной режим, но все остальные правила дорожного кодекса, обязаны соблюдать.  
CODE 3 - Срочный вызов. По этому коду принимаются вызовы, которые могут нести жертвы или идет угроза жизни окружающих. При данном коде, Вы выдвигаетесь на место с включенной сиреной и системой сликтоп (с сиреной и стробоскопами ). При данном коде игнорируются большинство правил дорожного кодекса, но Вы обязаны соблюдать свою полосу, а также никаких поездок не по проезжей части.  
CODE 4 - Помощь дополнительных юнитов не требуется. Этот код объявляется, когда Вы понимаете, что справитесь с ситуацией своими силами.  
CODE 4 ADAM - Поиск автомобиля/человека. Данный код объявляется тогда, когда во время погони, Вы теряете нарушителя из виду и начинаете его поиски. При объявлении данного кода, Вы обязаны указать предположительные районы, где может находиться нарушитель и указать, на сколько времени оглашается данный код.  
CODE 5 - Держаться на расстоянии от локации. Чаще всего данный код сообщается старшим или руководящим составами. При данном коде сообщается территорию,которую Вы обязаны игнорировать и не появляться на ней, даже если на эту территорию идут вызову 9-1-1.  
CODE 6 - Задерживаюсь на месте. Данный КОД означает, что Вы задерживаетесь на какой либо локации. Чаще всего объявляется когда Вы находитесь по вызову 9-1-1 или когда покидаете автомобиль при проверки какой либо местности. При данном КОДе Вы обязаны называть локацию на которой задерживаетесь.  
CODE 7 - Перерыв на обед. Данный код означает, что Вы берете перерыв ТОЛЬКО на прием пищи. При сообщении данного кода, Вы обязаны называть локацию где будете обедать и на какое время берете код. Максимальное время кода - это 15 минут.

1. Объявлять в розыск под прикрытием.
